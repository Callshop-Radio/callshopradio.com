# Security Policy

## Supported versions

| Branch | Supported |
| --- | --- |
| `main` | yes |

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security problems.

Email [support@callshopradio.com](mailto:support@callshopradio.com) with:

- A description of the issue and potential impact
- Steps to reproduce (proof-of-concept if available)
- Affected URLs, routes, or components

We aim to acknowledge reports within **3 business days** and will coordinate disclosure
timelines with you. Please allow time for a fix on `main` before public disclosure.

## Secret scanning

This repository uses GitHub secret scanning and push protection. If a push is blocked,
rotate the exposed credential immediately and contact us if you need help assessing impact.
